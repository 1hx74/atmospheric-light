Корпус состоит из трёх частей. Печать выполнена из пластика **Standard PETG** с толщиной слоя **0.2 мм**. Горизонтальная компенсация (horizontal expansion) при печати была установлена на **-0.07 мм**.

К сожалению, сборка проекта в данный корпус оказалась затруднена из-за недостатка опыта в 3D-моделировании — изначально не были учтены некоторые конструктивные особенности для удобной сборки. Тем не менее, сборка возможна. В дальнейшем корпус будет переработан с учётом этих недочётов.

### Использованные компоненты:

- Ползунковый энкодер

- Конденсатор 470 мФ, 6.3 В  
- Быстросъёмный DC-разъём  
- Разъём DS1110-01-4
