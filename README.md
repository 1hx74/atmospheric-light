# Atmospheric Light

Arduino-based garland project.

The assembly diagram and the source code of the early version are already available, the 3D model of the case will be available later.  
The system is designed so that new lighting effects can be easily added.

> ⚠️ Project is currently in development

## Guide

- Press once — next effect  
- Press twice — previous effect  
- Press three times — enter demo mode  
- Press once or twice — exit demo mode  

## UI Philosophy

The garland includes a simple built-in UI with its own design philosophy.

All service effects that are part of the interface use **white color**, which is chosen as the primary UI color.

For critical actions, a secondary accent color — **red** — is intended to be used.

> ⚠️ UI behavior and effects are still work in progress and may be inconsistent
